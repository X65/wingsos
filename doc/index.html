Congratulations!

You've managed to catch jolz64.dyn.dhs.org online, running JWEB Server on the SuperCPU C64 OS, JOS.

The specs for this machine are:

  • Commodore 64 + SuperCPU cartridge with 20Mhz Western Design Center 65816.
  • 2Mb RAM (Server+OS will run without extra RAM).
  • X1541 cable link to Linux machine emulating a 1541 disk drive.
  • PPP link using the X1541 cable.
  • Simple multithreaded webserver.
  • The JOS operating system.

    The Operating System specs:

  • Multitasking, multithreading small Microkernel. (~11K)
  • Very small TCP/IP implementation. (~4k)
  • Dynamically linked shared Libraries.
  • Code segment sharing.
  • Allows support for many different filesystems.
  • C library and 65816 backend for LCC compiler.
